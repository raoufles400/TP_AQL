
# TP3 – Tests d’Intégration Java avec Docker, Testcontainers et Spring Boot

## Objectif

Ce TP a pour objectif de vous initier aux tests d’intégration en Java en combinant plusieurs technologies modernes. Vous apprendrez à :

- Utiliser Docker pour faire tourner des services dans des conteneurs isolés
- Automatiser les tests d’intégration avec Testcontainers
- Mettre en place et tester une application Spring Boot qui communique avec une base de données MySQL
- Remplacer les tests classiques par des tests plus fiables et réalistes grâce à l’utilisation de conteneurs

## Contenu du TP

### Exercice 1 – Utilisation de Testcontainers

Objectifs :

- Remplacer les mocks par de véritables services déployés dans des conteneurs Docker
- Construire ou utiliser des images Docker pour les services nécessaires aux tests
- Écrire des tests d’intégration utilisant Testcontainers afin d’exécuter les dépendances dans un environnement propre et contrôlé

### Exercice 2 – Tests d’intégration d’une application de gestion de tâches

Objectifs :

- Télécharger ou cloner une application Java de gestion de tâches
- Réécrire les tests d’intégration existants en utilisant Testcontainers pour isoler les dépendances (comme la base de données)
- Exécuter les tests et vérifier leur fiabilité dans un environnement reproductible
